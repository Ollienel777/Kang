ClipFarm

Automated Volleyball Highlight Generator

Users upload raw game film; YOLOv8-pose extracts player skeletons frame-by-frame. A rule-based action classifier scores spikes, serves, digs, sets, and blocks. FFmpeg cuts and re-encodes the resulting clips to H.264, served via presigned Cloudflare R2 URLs.

Audio RMS analysis on extracted PCM weights detection confidence by crowd noise, reducing false-positive clips by ~20%. The backend uses FastAPI for auth-gated REST endpoints, Celery + Redis for async video jobs, and Modal for GPU-offloaded YOLOv8 inference. Supabase Auth with ES256 JWKS verification handles per-user data isolation and Row-Level Security.
